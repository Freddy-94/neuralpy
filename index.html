Get Started

Installing neuralpy

$ pip install neuralpy

Tutorial: AND function


Let's start with a simple AND function, which takes in two inputs that can be 1 or 0 and returns 1 when both inputs are 1 and returns 0 otherwise.

After installing, import the package:


>>> import neuralpy

Now you can create an instance of the Network class and specify the structure of the network:

>>> import neuralpy
>>> net = neuralpy.Network([2, 3, 4, 1])

The code above creates a network with two input nodes, three nodes in the first hidden layer, four nodes in the second hidden layer, and two output nodes. Now you can start loading data in.

Inputs and outputs are stored as a list of tuples of vectors. You can grab a simple AND dataset by executing the following:

>>> dataset = neuralpy.load_and()

Details about the input/output types are in the docs. Now let's set the number of epochs and a learning rate and start training:

>>> epochs = 100
>>> learning_rate = 0.1
>>> net.train(dataset, epochs, learning_rate)

Let's verify that it worked:

>>> for x, y in dataset:
    output = float(net.forward(x))
    print "expected: " + str(y)
    print "got: " + str(output) + "\n"
